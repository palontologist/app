## Color Scheme (Based on Project Theme)

- **Primary Action**: Green (#28A745) - "Sync Google Calendar" button
- **Success**: Green text (#28A745) - Success messages
- **Error**: Red text (#DC3545) - Error messages
- **Icons**: Calendar icon from lucide-react
- **Border**: Subtle gray borders for event cards
- **Background**: White cards on light gray background

## Responsive Behavior

### Desktop (> 1024px)
- Button appears in header next to "Upcoming Events" title
- Success/error messages appear below button
- Events displayed in full-width cards

### Tablet (768px - 1024px)
- Button remains in header but may wrap on smaller screens
- Events displayed in slightly narrower cards

### Mobile (< 768px)
- Button may stack below title on very small screens
- Events displayed in mobile-optimized compact cards
- Touch-friendly tap targets
